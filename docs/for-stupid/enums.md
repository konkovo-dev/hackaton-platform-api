# Global Enums (v0)

## Hackathon Roles (hackathon-level)
> What it is: права доступа пользователя внутри конкретного хакатона (оргсостав), независимые от участия как участника.

| Enum | Description |
|---|---|
| `HX_ROLE_OWNER` | Создатель/владелец хакатона, полный контроль (включая управление организаторами). |
| `HX_ROLE_ORGANIZER` | Управляет хакатоном (контент, настройки, сообщения участникам), но не управляет организаторами. |
| `HX_ROLE_MENTOR` | Доступ к разделу/функциям ментора внутри хакатона. |
| `HX_ROLE_JUDGE` | Доступ к разделу/функциям жюри внутри хакатона. |

---

## Participant Status (HackathonUser / participation)
> What it is: текущее состояние участия пользователя в хакатоне (режим и привязка к команде).

| Enum | Description |
|---|---|
| `PART_NONE` | Пользователь не зарегистрирован на хакатон (нет HackathonUser/participation записи). |
| `PART_INDIVIDUAL_ACTIVE` | Участвует индивидуально (имеет доступ к Challenge/Submit как индивидуальный участник). |
| `PART_LOOKING_FOR_TEAM` | Зарегистрирован без команды и обязан вступить в команду до дедлайна, иначе станет неактивным. |
| `PART_TEAM_MEMBER` | Участвует в составе команды как участник. |
| `PART_TEAM_CAPTAIN` | Участвует в составе команды как капитан. |
| `PART_INACTIVE` | Участие неактивно (например, не вступил в команду до дедлайна или снят). |

## Invitation Type
> What it is: тип приглашения/запроса в едином механизме invitations.

| Enum | Description |
|---|---|
| `INV_ROLE_ASSIGNMENT` | Приглашение на хакатонную роль (например, ORGANIZER/MENTOR/JUDGE). |
| `INV_TEAM_JOIN_REQUEST` | Заявка кандидата на вступление в команду (адресат — капитан). |
| `INV_TEAM_INVITE` | Приглашение капитана кандидату вступить в команду (опционально, если используем). |

---

## Invitation Status
> What it is: жизненный цикл приглашения/запроса (что с ним произошло).

| Enum | Description |
|---|---|
| `INV_PENDING` | Ожидает решения получателя. |
| `INV_ACCEPTED` | Принято (дальше запускаются доменные действия: роль назначена/вступление в команду). |
| `INV_DECLINED` | Отклонено. |
| `INV_EXPIRED` | Истекло по времени. |
| `INV_CANCELED` | Отменено отправителем. |

## CTA Actions (BFF)
> What it is: “главные кнопки”/действия, которые BFF возвращает для текущего экрана и контекста.

| Enum | Description |
|---|---|
| `CTA_SIGNUP_OR_LOGIN` | Предложить пользователю зарегистрироваться/войти в систему. |
| `CTA_REGISTER` | Перейти к регистрации на хакатон. |
| `CTA_OPEN_ME_IN_HACKATHON` | Открыть раздел “Me in hackathon” (мой контекст/анкета/команда/заявки). |
| `CTA_OPEN_CHALLENGE` | Открыть раздел “Challenge & Submit”. |
| `CTA_OPEN_ORGANIZER` | Открыть Organizer-панель. |
| `CTA_OPEN_MENTOR` | Открыть Mentor-раздел. |
| `CTA_OPEN_JUDGE` | Открыть Judge-раздел. |

## Sidebar Sections
> What it is: секции левого меню внутри хакатона (BFF решает, какие доступны).

| Enum | Description |
|---|---|
| `SEC_MAIN` | Главная страница хакатона (общая информация). |
| `SEC_ME_IN_HACKATHON` | “Me in hackathon”: анкета участия + командный блок + заявки/инвайты. |
| `SEC_CHALLENGE_SUBMIT` | “Challenge & Submit”: задание, сабмит, сообщения организаторов. |
| `SEC_ORGANIZER` | Панель организатора (управление хакатоном и сообщениями). |
| `SEC_MENTOR` | Раздел ментора. |
| `SEC_JUDGE` | Раздел жюри. |

---

## Registration Policy (hackathon)
> What it is: какие режимы регистрации разрешены в конкретном хакатоне.

| Enum | Description |
|---|---|
| `REG_ALLOW_INDIVIDUAL` | Разрешено индивидуальное участие. |
| `REG_ALLOW_TEAM` | Разрешено командное участие (создание команды и/или “найду команду позже”). |
